# Speed Informática SM

Landing page estática para publicação no GitHub Pages.

## Estrutura

- `index.html`: página principal
- `styles.css`: identidade visual
- `script.js`: ano automático no rodapé
- `.nojekyll`: evita processamento do GitHub Pages
- `404.html`: fallback para rotas inválidas

## Publicação no GitHub Pages

1. Suba estes arquivos para o repositório.
2. No GitHub, vá em `Settings` > `Pages`.
3. Selecione a origem da publicação:
   - `Deploy from a branch`, usando a branch principal e a pasta raiz, ou
   - `GitHub Actions`, se você quiser automatizar depois.
4. Aguarde o GitHub gerar a URL pública.
